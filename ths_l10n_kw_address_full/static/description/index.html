Kuwait Address Full

Enhanced Address Localization for Kuwait

This module provides comprehensive address management for Kuwait, aligning Odoo's standard address fields with the local geographical hierarchy of Governorates and Areas/Districts. It populates essential data and customizes address entry and display to meet Kuwaiti standards.

Key Features

Kuwaiti Geographical Data

  • Populates all 6 Kuwaiti Governorates (`محافظة`) into Odoo's standard `States` (`res.country.state`).
  • Introduces a new `Area/District` model (`res.country.area`) to manage the detailed areas within each Governorate.
  • Provides an extensive initial list of common Areas/Districts for Kuwait.

Smart Address Entry

  • **Re-purposes the standard 'City' field** on `res.partner` to be a `Many2one` dropdown for 'Area / District'.
  • Dynamically filters 'Area / District' options based on the selected 'Governorate' (Odoo's `State`).
  • Automatically sets the 'Governorate' and 'Country' when an 'Area / District' is selected.
  • Automatically clears 'Area / District' if the selected 'Governorate' or 'Country' changes.

Customized Address Formatting

  • Modifies the default address display format to:
    %(street)s
    %(street2)s
    %(city)s %(state_code)s
    %(country_name)s
    This will render as: `[Street]`, `[Street2]`, `[Area/District] [Governorate Code]`, `[Country Name]`.
  • Hides the 'Zip' (Postal Code) field from address forms as it's not commonly used for specific street addresses in Kuwait.

Improved Backend Visibility

  • Adds an 'Areas / Districts' column to the States tab within the Country configuration form, allowing easy overview of areas per Governorate.

Configuration & Usage

Partner/Company Addresses: When creating or editing a partner's address, you will now select the 'Country', then the 'Governorate' (from the State dropdown), and finally the 'Area / District' (from the City dropdown).

Managing Areas: You can view and manage all defined Areas/Districts under `Settings -> Technical -> Database Structure -> Countries -> Areas / Districts`.

Country States View: Navigate to `Settings -> General Settings -> Countries` and open 'Kuwait'. In the 'States' tab, you will now see a column listing the 'Areas / Districts' linked to each Governorate.
