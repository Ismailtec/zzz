Company to Branch Terminology Transformation

This module transforms Odoo's standard "Company" terminology to "Branch" across all key accounting models, views, and reports. Perfect for organizations that prefer branch-based thinking while maintaining full compatibility with Odoo's native multi-company functionality. Covers invoices, payments, reports, and all accounting workflows.

Fully Compatible with Odoo 18's native multi-company features and all standard accounting workflows. Simply changes terminology while preserving all functionality.

Installation & Usage

Installation: Simply install the module through Apps menu. No additional configuration required - all changes are applied automatically.

Branch Setup: Use the standard Companies menu to create and manage your branches. All existing multi-company functionality works exactly the same.

User Access: Assign users to branches using the standard company assignment. The branch switcher works just like the company switcher.

Reports: All accounting reports now show "Branch" filters instead of "Company". Filter by single or multiple branches as needed.
